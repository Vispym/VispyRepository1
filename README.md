# Game API

A simple ASP.NET Core Web API for managing a list of games.

## Features
- Get all games
- Get a game by ID
- Create a new game

## Run locally
```bash
dotnet run
```

Then open:
- http://localhost:5142/
- http://localhost:5142/api/games

## Example endpoints
- GET /api/games
- GET /api/games/1
- POST /api/games
